# 第七章　铁心 — 分镜脚本

## 场景一：多里安的公开广播

**时间**：第五天清晨 6:03
**地点**：城市各处——卧室、街道、公共空间
**时长估算**：约4分钟
**氛围**：震惊、凝固、爆发

| **镜头目的** | 建立"最后的正常时刻"——这是多里安广播前的最后一秒宁静 |

| **光影** | 卧室的晨光突然变得刺眼，广播中断时灯光闪烁了一下 |

| **对白** | 多里安："早上好，各位公民。我是多里安。你们中的大多数人可能从未听过我的名字。但你们一定听过弥赛亚的名字。" |
| **镜头目的** | 多里安声音的突然入侵——打破日常的第一声惊雷 |

## 场景五：血墙与陈小鱼

**时间**：紧接上一场景
**地点**：废弃厂房内部深处
**时长估算**：约3分钟
**氛围**：恐惧、震撼、意志的试炼

### 镜头 07-26

| 项目 | 内容 |
|------|------|
| **景别** | 苏铭哲背影中景 → 急速推进至墙壁特写 |
| **镜头运动** | 从苏铭哲的背影中景开始，随着他的视线急速推进至墙壁 |
| **画面内容** | 苏铭哲停下了脚步。他的右脚刚刚抬起，悬在半空中，然后缓缓放下，鞋底接触地面时没有发出任何声音——他本能地放轻了脚步。他的身体微微前倾，肩膀向内收拢，头部微微抬起，目光定格在走廊尽头的墙壁上。镜头从他的背影中景开始，随着他的视线急速推进至墙壁——那面墙。那面墙上写满了字。不是打印的——没有打印机可以在这种环境下工作。不是投影的——没有任何投影设备的痕迹。是用某种深红色的液体写的。字迹潦草而急促，笔画中带着颤抖的痕迹，有些地方的液体还在缓缓流淌，在灰色的混凝土墙壁上拖出长长的痕迹，像一道道正在流血的伤口。字体的大小不一——最大的字有半米高，最小的字只有拳头大小，它们杂乱无章地分布在整面墙壁上，从左下角一直延伸到右上角，像一场疯狂的书写留下的遗迹。镜头急速推进至墙壁的中心位置——那里写着一行字，字体比其他的都大，笔画比其他的都粗，液体比其他的都浓："你的意志——还在吗？"那行字的最后一个问号的尾巴拖得很长，向下延伸了大约三十厘米，像一滴正在坠落的血珠 |
| **光影** | 应急灯的惨淡光线照亮墙壁，深红色的字迹在灰色混凝土上显得格外刺目，液体的反光令人不安 |
| **音效** | 苏铭哲停止脚步的声响、液体缓缓流淌的微弱声音、他骤然加重的呼吸声 |
| **镜头目的** | 血墙的出现——多里安的"拷问"从数字世界延伸到了物理世界 |

### 镜头 07-27

| 项目 | 内容 |
|------|------|
| **景别** | 大特写 |
| **镜头运动** | 缓慢推进，聚焦于墙壁上的字迹细节 |
| **画面内容** | 墙壁上的字迹细节。镜头以极近的距离拍摄那些深红色的字迹——近到可以看清液体的每一个细节。那些深红色的液体——不是颜料，颜料会有均匀的质地和一致的颜色。这是血。血的颜色会随着时间而变化——刚流出的血是鲜红色的，氧化后的血是暗红色的，干涸的血是深褐色的。这面墙上的字迹包含了所有三种颜色：最新写的那几个字还是鲜红色的，液体的表面在应急灯的光线下微微反光，像一层薄薄的油膜；写得较早的字已经变成了暗红色，液体已经停止流动，在笔画的边缘形成了一圈深色的轮廓；写得最早的字已经干涸，变成了深褐色，与灰色的混凝土墙壁几乎融为一体，只有在特定的角度下才能辨认出来。字迹的笔画中带着颤抖的痕迹——那不是故意的艺术效果，而是书写者的手在颤抖。每一笔的起笔处都有一个小墨点，那是笔尖在墙壁上停留了一秒的结果——书写者在每一笔开始前都在犹豫，但最终还是写下去了。有些地方的液体已经干涸，有些地方还在缓缓流淌，流淌的速度很慢，大约每秒钟移动一毫米，像时间本身在这一刻变得粘稠了 |
| **光影** | 血迹在应急灯的光线下呈现出暗红色的光泽，干涸的部分变成深褐色，流淌的部分还在微微反光 |
| **音效** | 血液滴落的微弱声响、苏铭哲瞳孔收缩时的主观心跳声 |
| **镜头目的** | 血字的细节冲击——这不是威胁，这是多里安对"意志"概念的终极追问 |

### 镜头 07-28

| 项目 | 内容 |
|------|------|
| **景别** | 苏铭哲面部大特写 |
| **镜头运动** | 急速推进至苏铭哲的面部大特写 |
| **画面内容** | 苏铭哲停在那面墙前，距离墙壁大约两米。镜头急速推进至他的面部大特写——近到可以看清他脸上的每一个毛孔、每一根睫毛、每一个微表情的变化。他的瞳孔骤然收缩——收缩的幅度如此之大，以至于他的眼睛在那一瞬间看起来像两个黑色的针尖，针尖的周围是一圈深褐色的虹膜，虹膜的纹路在收缩中变得格外清晰。他的嘴唇微微张开，但没有发出声音——下颌的肌肉僵硬了，声带在喉咙中绷紧，但没有振动。他的面部肌肉整体变得僵硬，像一尊正在风化的石像。他的大脑在疯狂地处理眼前的信息——血写的字、化学制剂的甜腻气味、多里安的"实验"、陈小鱼的安危、通讯器上的那句"你的意志——还在吗？"——所有的线索在这一刻汇聚成一个令人窒息的真相：多里安不是在威胁他，不是在恐吓他，不是在折磨他。多里安是在问他一个问题——一个关于他苏铭哲本人的问题。你的意志还在吗？你还剩下多少？你能承受多少？你会在什么时候崩溃？这些问题像碎片一样在他的大脑中旋转、碰撞、重组，试图拼凑出一个答案，但那个答案太大了、太沉了，他暂时无法把它从喉咙里挤出来 |
| **光影** | 血墙的暗红色光芒映照在苏铭哲的脸上，他的瞳孔中反射着那些血字 |
| **音效** | 苏铭哲急促的心跳声、他压抑的呼吸声、远处传来微弱的呜咽声 |
| **镜头目的** | 苏铭哲的内心崩溃与重建——他正在经历多里安所说的"意志"的考验 |

### 镜头 07-29

| 项目 | 内容 |
|------|------|
| **景别** | 通讯器特写 → 苏铭哲中景 |
| **镜头运动** | 从通讯器特写开始，拉远至苏铭哲的中景 |
| **画面内容** | 苏铭哲的通讯器再次亮了。他低头看向外套口袋——口袋里的通讯器正在发出冷蓝色的光芒。他将通讯器从口袋中掏出来，屏幕上显示的不是文字，而是一个来电提示——发信人：凯文。他犹豫了一秒——只有一秒——然后接通了。凯文的声音从通讯器中传出，急促而清晰："苏铭哲，你在哪？我们已经锁定了区域——"声音很大，在空旷的走廊中回荡。但苏铭哲已经听不进去了——不是听不见，是听不进去。凯文的声音像水滴落在石头上，从他的意识表面滑落，没有留下任何痕迹。他的目光越过血墙，看向走廊的深处——那里有微弱的光线，比应急灯的光线更亮，更温暖，像是从某个房间中透出来的。那里有微弱的声音——不是金属的吱嘎声，不是水滴的滴落声，而是一种更柔软的、更人性的声音。那是陈小鱼的声音。她还在那里。她还活着。苏铭哲将通讯器塞回口袋——没有挂断，只是塞回去，让凯文的声音在口袋里继续回荡。然后他迈开脚步，朝那个方向走去。每一步都比上一步更坚定，每一步都比上一步更沉重，每一步都比上一步更无法回头 |
| **光影** | 通讯器的冷蓝色光芒与走廊深处的微弱光线形成对比，苏铭哲选择了更深的黑暗 |
| **音效** | 凯文的声音从通讯器中传出、苏铭哲的脚步声再次响起、走廊深处微弱的声音 |
| **镜头目的** | 苏铭哲做出了选择——他不再等待增援，而是独自走向未知 |

### 镜头 07-30

| 项目 | 内容 |
