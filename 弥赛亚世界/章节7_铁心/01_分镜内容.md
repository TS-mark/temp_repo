# 第七章　铁心 — 分镜脚本

## 场景一：多里安的公开广播

**时间**：第五天清晨 6:03
**地点**：城市各处——卧室、街道、公共空间
**时长估算**：约4分钟
**氛围**：震惊、凝固、爆发

| 项目 | 内容 |
|------|------|
| **景别** | 大远景 |
| **镜头运动** | 固定机位，从城市天际线远景拍摄 |
| **画面内容** | 清晨六点零三分的城市。天色微亮，银白色的建筑群在晨光中泛着冷光。一切如常——路灯正在关闭，早起的行人开始出现在街道上，飞行器在空中有序穿行。弥赛亚的公共广播系统正在播放惯常的晨间新闻，温和的女声从每一个扬声器中传出 |
| **光影** | 晨光的冷蓝色调，城市灯光逐渐熄灭，自然光接管一切 |
| **音效** | 弥赛亚广播的温和女声，城市苏醒的日常声音——脚步声、飞行器引擎声、电子嗡鸣 |
| **镜头目的** | 建立"最后的正常时刻"——这是多里安广播前的最后一秒宁静 |
