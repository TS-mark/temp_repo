# 第九章　完美世界 — 分镜脚本

## 场景一：凯文消失的三天

**时间**：顾远山牺牲后的清晨 → 第四天傍晚
**地点**：秩序课驻地 → 城市各处
**时长估算**：约2分钟
**氛围**：焦灼的追踪、倒计时的紧迫

---

### 镜头 09-01-01

| 项目 | 内容 |
|------|------|
| **景别** | 中景（空荡的床铺） |
| **镜头运动** | 固定机位，静止拍摄，长时间不动 |
| **画面内容** | 清晨的光线透过窗户照进驻地的宿舍。凯文的床铺——被褥整齐地叠好，枕头放在正中，但人已经不在了。床头柜上没有留言，没有便条，什么都没有。镜头保持不动，让空荡的床铺填满画面——那种"人已经走了"的空白感比任何文字都更有说服力 |
| **光影** | 清晨的冷白色光线照在空床上，被褥的褶皱在光中形成浅淡的阴影 |
| **音效** | 驻地清晨的环境音——远处有人走动的脚步声、通风系统的嗡鸣。但这个房间里是安静的 |
| **旁白** | （旁白，男声低沉）"凯文消失了三天。在顾远山牺牲后的那个清晨，他从驻地消失了。没有告别，没有留言，没有任何征兆——就像他两年来无数次独自追踪多里安时一样，无声无息地融入了城市的阴影中。" |
| **镜头目的** | 用空床建立"缺席"——凯文的消失不需要解释，观众已经知道他去了哪里 |

## 场景二：穹顶温室

**时间**：第四天傍晚 / 太阳正在落山
**地点**：弥赛亚废弃农业研究中心 — 穹顶温室
**时长估算**：约2分钟
**氛围**：末世的神圣感、被遗忘的美、时间的停滞

---

### 镜头 09-02-01

| 项目 | 内容 |
|------|------|
| **景别** | 大远景（穹顶外景） |
| **镜头运动** | 从地面仰拍穹顶，缓慢推进至穹顶的玻璃表面 |
| **画面内容** | 穹顶温室的外景。直径三百米的玻璃穹顶矗立在废弃的农业研究中心中央，表面布满裂纹和尘垢，但整体结构依然完整。夕阳的光线从穹顶的一侧照入，碎裂的玻璃将光线分解成无数细碎的光柱，穿过穹顶内部的空间。穹顶周围的建筑已经荒废，杂草丛生，但穹顶本身像一座被时间遗忘的神殿 |
| **光影** | 夕阳的金红色光线从穹顶西侧的碎裂处涌入，形成大面积的光柱。穹顶的东侧处于阴影中，玻璃上的裂纹在逆光中形成蛛网般的金色线条 |
| **音效** | 风穿过穹顶碎裂处的呜咽声，远处金属结构在温差变化中的轻微吱嘎声 |
| **镜头目的** | 建立穹顶温室的视觉规模——三百米的直径意味着它是一个微型世界 |
