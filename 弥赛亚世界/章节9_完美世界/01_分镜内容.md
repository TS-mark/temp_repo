# 第九章　完美世界 — 分镜脚本

## 场景一：凯文消失的三天

**时间**：顾远山牺牲后的清晨 → 第四天傍晚
**地点**：秩序课驻地 → 城市各处
**时长估算**：约2分钟
**氛围**：焦灼的追踪、倒计时的紧迫

---

### 镜头 09-01-01

| 项目 | 内容 |
|------|------|
| **景别** | 中景（空荡的床铺） |
| **镜头运动** | 固定机位，静止拍摄，长时间不动 |
| **画面内容** | 镜头从宿舍的窗户开始，清晨的冷白色光线透过半透明的窗帘洒进来，在地面上形成一片柔和的光斑。镜头缓缓向房间内部推进，首先映入眼帘的是靠窗的书桌——桌面上空无一物，只有一层薄薄的灰尘在光线中微微闪烁。镜头继续向右移动，对准了房间中央的床铺。被褥被整齐地叠成了标准的方块形状，每一个褶皱都被仔细抚平，枕头被端正地放在床头的正中央。床单的表面平整得没有一丝皱褶，仿佛昨晚根本没有人在这张床上睡过。镜头在这个画面上停留了整整三秒，没有任何移动，没有任何声音——只有清晨的光线静静地照在这张空荡荡的床上，那种"人已经走了"的空白感比任何文字都更有说服力 |
| **光影** | 清晨的冷白色光线照在空床上，被褥的褶皱在光中形成浅淡的阴影 |
| **音效** | 驻地清晨的环境音——远处有人走动的脚步声、通风系统的嗡鸣。但这个房间里是安静的 |
| **旁白** | （旁白，男声低沉）"凯文消失了三天。在顾远山牺牲后的那个清晨，他从驻地消失了。没有告别，没有留言，没有任何征兆——就像他两年来无数次独自追踪多里安时一样，无声无息地融入了城市的阴影中。" |
| **镜头目的** | 用空床建立"缺席"——凯文的消失不需要解释，观众已经知道他去了哪里 |

**氛围**：末世的神圣感、被遗忘的美、时间的停滞
