## 场景八：艺术学院调查

**时间**：标本案件简报后 / 白天
**地点**：东辰艺术学院
**时长估算**：约4分钟
**氛围**：诡异、压抑、隐藏的危险

---

### 镜头 08-01

| 项目 | 内容 |
|------|------|
| **景别** | 大远景 → 缓慢推进 |
| **镜头运动** | 航拍，从高空缓慢下降，穿过人造森林的树冠 |
| **画面内容** | 东辰艺术学院坐落在城市的东北角，被一片人造森林环绕。校门是两棵交缠的金属树，树冠由无数细小的光纤编织而成，在阳光下折射出彩虹般的光芒。苏铭哲和凯文并肩走进校园。苏铭哲注意到凯文今天没有带书——他把那本《华氏451》留在了办公室 |
| **光影** | 阳光透过人造森林的树叶洒下斑驳的光影，金属树校门的光纤树冠折射出七彩光芒 |
| **音效** | 人造森林中树叶摩擦的沙沙声，远处学生们的交谈声，脚步踩在石板路上的轻响 |
| **镜头目的** | 建立艺术学院的空间感——美丽、精致、与外界的压抑形成对比 |

### 镜头 08-02

| 项目 | 内容 |
|------|------|
| **景别** | 中景 |
| **镜头运动** | 跟拍两人行走，缓慢环绕 |
| **画面内容** | 苏铭哲和凯文并肩走在校园的小径上。苏铭哲问起布拉德伯里，凯文的目光扫过校园里的学生，语气平淡地回答。他的眼睛在每一个学生身上停留的时间都很短，但苏铭哲注意到他在扫描——像猎人在寻找猎物 |
| **光影** | 树荫下的柔和光线，两人的影子在石板路上交叠 |
| **音效** | 脚步声的节奏，远处学生的笑声和交谈声 |
| **对白** | 苏铭哲："你读过布拉德伯里？" / 凯文："读过。蒙塔格用火焰焚烧书籍，以为消灭了文字就能消灭思想。但那是上个世纪的故事。在弥赛亚的世界里，不需要焚书——系统会在你产生危险念头之前就拦住你。" / 苏铭哲："如果拦不住呢？" / 凯文："那就说明系统有漏洞。而我的工作，就是堵上这些漏洞。" |
| **镜头目的** | 凯文的哲学观——他不是弥赛亚的信徒，而是务实的执行者 |

### 镜头 08-03

| 项目 | 内容 |
|------|------|
| **景别** | 中景 |
| **镜头运动** | 正反打，凯文与学院负责人之间 |
| **画面内容** | 他们找到了学院的负责人——一个笑容可掬的中年女人，危险值稳定在12，几乎是完美的数字。她热情地介绍学院的情况，但当凯文提到标本事件时，她的笑容明显僵硬了一瞬。凯文的目光锐利，捕捉到了那一瞬间的变化。负责人犹豫后调出了一份名单 |

| **音效** | 负责人说话时的热情语调，凯文提到标本事件时的短暂沉默，数据终端调出名单的电子音 |
| **对白** | 负责人："那件事……我们已经全力配合过调查了。" / 凯文："我们需要重新走访。特别是那些在案发时间段内表现异常的学生。" / 负责人："有一个学生……我不能说她'异常'，但她确实比较……特殊。" / 凯文："谁？" / 负责人："林雨桐。油画系三年级。" |
| **镜头目的** | 引出林雨桐——负责人的犹豫暗示这个学生不同寻常 |

### 镜头 08-04

| 项目 | 内容 |
|------|------|
| **景别** | 中景 → 缓慢环视 |
| **镜头运动** | 跟随苏铭哲的脚步进入画室，缓慢环视四面墙壁 |
| **画面内容** | 他们被带到了三楼的一间画室。门推开的瞬间，苏铭哲闻到了松节油和亚麻籽油的气味——真正的颜料，不是数字模拟的那种。画室里没有人，但四面墙壁上挂满了画。苏铭哲一幅一幅地看过去，脚步越来越慢。第一幅：一个透明的人形，内部的骨骼和脏器清晰可见，跪在一面破碎的镜子前。第二幅：一片荒原上，无数人形如稻草般矗立，每个人的胸口都亮着一盏灯。第三幅：一只手从地底伸出，掌心里托着一只眼睛 |
| **光影** | 画室的自然光从天窗洒入，松节油的气味在光线中几乎可见，画作的色彩在光线下鲜艳而诡异 |
| **音效** | 门推开的轻微吱嘎声，脚步在画室地板上的回响，松节油气味带来的嗅觉暗示（画面中可加入轻微的空气流动感） |
| **镜头目的** | 展示林雨桐的画作——技法精湛但内容诡异，暗示她的内心世界 |

| **镜头目的** | 画作的象征意义——每一幅都暗示着弥赛亚世界的某种真相 |

| **镜头运动** | 苏铭哲转身的动作，快速切至门口的女孩 |
| **画面内容** | "你们在看我的画？"苏铭哲猛地转身。门口站着一个女孩。瘦小，安静，一头黑色的长发垂到腰际。她穿着学院的制服，手里抱着一块画板，脸上的表情淡漠得像一潭死水。凯文问她的名字，她点了点头。苏铭哲偷偷扫了一眼她手腕上的植入芯片——危险值：15。完美的数字。低得无可挑剔 |
