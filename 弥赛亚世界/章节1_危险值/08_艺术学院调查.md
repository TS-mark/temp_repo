## 场景八：艺术学院调查

**时间**：标本案件简报后 / 白天
**地点**：东辰艺术学院
**时长估算**：约4分钟
**氛围**：诡异、压抑、隐藏的危险

---

### 镜头 08-01

| 项目 | 内容 |
|------|------|
| **景别** | 大远景 → 缓慢推进 |
| **镜头运动** | 航拍，从高空缓慢下降，穿过人造森林的树冠 |
| **画面内容** | 镜头从高空俯瞰，城市的东北角有一片明显的绿色——人造森林，由基因工程培育的速生树木组成，树冠修剪得整齐划一，像一块铺在灰色城市中的绿色地毯。镜头缓缓下降，穿过树冠的顶层，阳光透过树叶的缝隙洒下来，在镜头前形成一道道金色的光柱。树冠之下，东辰艺术学院的建筑群逐渐显露：几栋低矮的白色建筑散布在森林之中，建筑的外墙覆盖着一层哑光的白色涂层，线条简洁而优雅，与城市中银白色的弥赛亚建筑风格截然不同。镜头继续下降，来到校门前——校门是两棵交缠的金属树，树干由不锈钢铸造，表面经过了哑光处理，树冠由无数细小的光纤编织而成，每一根光纤都在阳光下折射出不同颜色的光芒，整个树冠看起来像一团凝固的彩虹。苏铭哲和凯文并肩走进校门，苏铭哲穿着深蓝色的秩序课制服，步伐略显拘谨；凯文走在他右侧，步伐散漫而随意，左手插在裤兜里，右手垂在身侧——苏铭哲注意到凯文今天没有带书，那本《华氏451》被留在了办公室的桌面上 |
| **光影** | 阳光透过人造森林的树叶洒下斑驳的光影，金属树校门的光纤树冠折射出七彩光芒 |
| **音效** | 人造森林中树叶摩擦的沙沙声，远处学生们的交谈声，脚步踩在石板路上的轻响 |
| **镜头目的** | 建立艺术学院的空间感——美丽、精致、与外界的压抑形成对比 |

| **画面内容** | 苏铭哲和凯文并肩走在校园的小径上，小径由灰白色的石板铺成，石板的缝隙里长出了细细的青苔。小径的两侧是修剪整齐的灌木丛，灌木丛后面是几栋教学楼，白色的外墙在阳光下显得格外刺眼。苏铭哲的视线扫过校园——远处的草坪上，几个学生坐在树下看书或画画，姿态悠闲而从容；教学楼的走廊里，偶尔有学生走过，步伐轻快，脸上挂着微笑，每一个人看起来都像是模范公民的模板。苏铭哲开口问起布拉德伯里，凯文的目光没有停留在苏铭哲身上，而是扫过校园里的每一个学生——他的眼睛在每一个学生身上停留的时间都很短，不到一秒，但苏铭哲注意到他的目光移动的轨迹是有规律的，从左到右，从近到远，像一台正在扫描的雷达。凯文的语气平淡，声音不高不低，但苏铭哲能听出那平淡底下压着的某种东西——不是愤怒，不是悲伤，更像是一种已经见过太多黑暗之后的疲惫 |
| **光影** | 树荫下的柔和光线，两人的影子在石板路上交叠 |
| **音效** | 脚步声的节奏，远处学生的笑声和交谈声 |
| **对白** | 苏铭哲："你读过布拉德伯里？" / 凯文："读过。蒙塔格用火焰焚烧书籍，以为消灭了文字就能消灭思想。但那是上个世纪的故事。在弥赛亚的世界里，不需要焚书——系统会在你产生危险念头之前就拦住你。" / 苏铭哲："如果拦不住呢？" / 凯文："那就说明系统有漏洞。而我的工作，就是堵上这些漏洞。" |
| **镜头目的** | 凯文的哲学观——他不是弥赛亚的信徒，而是务实的执行者 |

### 镜头 08-03

|------|------|
| **景别** | 中景 |
| **镜头运动** | 正反打，凯文与学院负责人之间 |
| **画面内容** | 他们找到了学院的负责人——一个笑容可掬的中年女人，穿着一件浅灰色的职业套装，头发盘成一个整齐的发髻，脸上挂着一种经过训练的职业微笑，嘴角的弧度恰到好处，露出八颗牙齿。她站在学院行政楼的走廊里，双手交叠在身前，手腕上的植入芯片微微发光——苏铭哲扫了一眼，危险值：12，几乎是完美的数字。她热情地介绍学院的情况，声音甜美而流畅，像在播放一段预录的宣传视频。当凯文提到标本事件时，她的笑容明显僵硬了一瞬——嘴角的弧度下降了一两毫米，眼周的肌肉收紧了一下，然后迅速恢复了正常，整个过程不到半秒。但凯文的目光锐利得像一把手术刀，他捕捉到了那一瞬间的变化，琥珀色的瞳孔微微收缩了一下。负责人犹豫了几秒，嘴唇微微张开又合上，最终还是转身走向身后的数据终端，手指在屏幕上划了几下调出了一份名单 |
| **光影** | 学院走廊的明亮灯光，负责人的笑容在灯光下显得过于完美 |
| **音效** | 负责人说话时的热情语调，凯文提到标本事件时的短暂沉默，数据终端调出名单的电子音 |
| **对白** | 负责人："那件事……我们已经全力配合过调查了。" / 凯文："我们需要重新走访。特别是那些在案发时间段内表现异常的学生。" / 负责人："有一个学生……我不能说她'异常'，但她确实比较……特殊。" / 凯文："谁？" / 负责人："林雨桐。油画系三年级。" |
| **镜头目的** | 引出林雨桐——负责人的犹豫暗示这个学生不同寻常 |

### 镜头 08-04

| 项目 | 内容 |
|------|------|
| **景别** | 中景 → 缓慢环视 |
| **镜头运动** | 跟随苏铭哲的脚步进入画室，缓慢环视四面墙壁 |
| **画面内容** | 他们被带到了三楼的一间画室。画室的门是木质的，表面刷了一层白色的漆，漆面上有几道被颜料溅到的痕迹——红的、蓝的、黄的，像抽象画的碎片。苏铭哲伸手推门，门轴发出一声轻微的吱嘎声。门推开的瞬间，一股浓烈的松节油和亚麻籽油的气味扑面而来——真正的颜料，不是数字模拟的那种，是用植物油和矿物粉调出来的、带着温度和质感的颜料。画室里没有人，但四面墙壁上挂满了画——油画，用真正的画布和真正的颜料画出来的油画。苏铭哲的脚步放慢了，他一幅一幅地看过去。第一幅：一个透明的人形，没有皮肤，内部的骨骼和脏器清晰可见——心脏在胸腔里、肺叶在两侧展开、肠道在腹腔中盘旋——跪在一面破碎的镜子前，镜子里映出的是一个完整的、穿着体面的人影。第二幅：一片灰白色的荒原上，无数人形如稻草般矗立，每个人的胸口都亮着一盏灯——有的蓝、有的红、有的已经熄灭。第三幅：一只手从地底伸出，五指张开，掌心里托着一只眼睛，眼睛的虹膜里倒映着一座城市的轮廓 |
| **光影** | 画室的自然光从天窗洒入，松节油的气味在光线中几乎可见，画作的色彩在光线下鲜艳而诡异 |
| **音效** | 门推开的轻微吱嘎声，脚步在画室地板上的回响，松节油气味带来的嗅觉暗示（画面中可加入轻微的空气流动感） |
| **镜头目的** | 展示林雨桐的画作——技法精湛但内容诡异，暗示她的内心世界 |

### 镜头 08-05

| 项目 | 内容 |

| **镜头运动** | 缓慢平移，依次展示三幅画作的细节 |

| **光影** | 画作颜料的质感在特写中清晰可见，松节油和亚麻籽油的真实颜料光泽 |
| **音效** | 几乎无声，仅有苏铭哲放慢的呼吸声 |
| **镜头目的** | 画作的象征意义——每一幅都暗示着弥赛亚世界的某种真相 |

### 镜头 08-08

| 项目 | 内容 |
