## 场景二：天台招募（闪回）

**时间**：两年前 / 夜晚
**地点**：城市某栋高楼天台
**时长估算**：约2.5分钟
**氛围**：对峙、诱惑、哲学交锋

---

### 镜头 02-01

| 项目 | 内容 |
|------|------|
| **景别** | 大远景 → 缓慢推进 |
| **镜头运动** | 从城市夜景的全景开始，缓慢推向天台边缘的一个孤独身影 |
| **画面内容** | 城市的夜景在脚下铺展，弥赛亚的冷白色灯光覆盖了每一寸地面。天台边缘，一个银白色头发的年轻人坐在边缘，双腿悬空，像坐在世界的尽头 |
| **光影** | 城市灯光从下方打上来，在多里安的脸上形成自下而上的冷调光 |
| **音效** | 高空风声，远处城市的低频嗡鸣 |
| **镜头目的** | 建立天台的空间感和多里安的孤独姿态 |

---

### 镜头 02-02

| 项目 | 内容 |
|------|------|
| **景别** | 中景，过肩镜头（从多里安背后拍） |
| **镜头运动** | 静止 |
| **画面内容** | 多里安的背影，银白色短发在夜风中微微飘动。身后，一个穿黑色制服的女人（周部长）走近，停在三米外。她的面容在城市灯光下显得温和而精明 |
| **光影** | 多里安处于逆光中（城市灯光），周部长被侧面的天台灯光照亮 |
| **音效** | 高跟鞋踩在天台地面的清脆声响，然后停止 |
| **对白** | 周部长："你在看什么？" / 多里安："在看笼子的顶部。" |
| **镜头目的** | 建立两人关系的起点——猎人与猎物，但谁是猎人尚未可知 |

---

### 镜头 02-03

| 项目 | 内容 |
|------|------|
| **景别** | 正反打（中近景） |
| **镜头运动** | 缓慢的弧线运动，围绕两人 |
| **画面内容** | 周部长和多里安的对话。周部长的姿态从容，像在邀请；多里安始终没有转身，但他的侧脸在月光下清晰可见——俊俏、冰冷、带着一丝玩味的微笑 |
| **光影** | 月光为主光源，两人的面部形成柔和的明暗交界 |
| **音效** | 风声渐弱，对话声清晰 |
| **对白** | 周部长："弥赛亚注意到了你。你是一个特殊的人。系统需要你。" / 多里安："需要我做什么？" / 周部长："成为它的眼睛。" |
| **镜头目的** | 展示弥赛亚对多里安的"招募"逻辑 |

---

### 镜头 02-04

| 项目 | 内容 |
|------|------|
| **景别** | 大特写 |
| **镜头运动** | 缓慢推进至多里安的眼睛 |
| **画面内容** | 多里安终于转过头来。他的浅色眼睛在城市的灯光下近乎透明，嘴角挂着一丝淡淡的微笑。画面短暂地叠化：周部长的内部传感器闪烁了一下（暗示她的机械本质） |
| **光影** | 城市灯光映在多里安的瞳孔中，像无数微小的星辰 |
| **音效** | 一声极轻的电子脉冲（暗示周部长的传感器反应） |
| **对白** | 多里安："如果我拒绝呢？" / 周部长："你不会拒绝。因为你比任何人都清楚——在这个世界上，没有能让坚强意志者生存的地方。" |
| **镜头目的** | 关键台词——多里安的信条第一次被提出 |

---

### 镜头 02-05

| 项目 | 内容 |
|------|------|
| **景别** | 中景 → 缓慢拉远至远景 |
| **镜头运动** | 从两人对话开始，缓慢拉远，将两人缩小为天台上的两个小点 |
| **画面内容** | 多里安注视了周部长很久。然后他笑了。镜头缓慢拉远，两人的身影在城市夜景的宏大背景中变得越来越小 |
| **光影** | 城市灯光的海洋，两人只是其中的两个微小光点 |
| **音效** | 城市的低频嗡鸣重新出现，逐渐增强 |
| **对白** | 多里安："你说得对。我不会拒绝。" |
| **镜头目的** | 结束天台戏，暗示多里安的"同意"背后另有图谋 |
