## 场景三：三个月潜伏与真相发现（闪回）

**时间**：两年前 / 三个月的时间跨度
**地点**：弥赛亚总部第七层
**时长估算**：约2分钟（蒙太奇）
**氛围**：伪装、渗透、逐渐揭示真相

---

### 镜头 03-01

| 项目 | 内容 |
|------|------|
| **景别** | 蒙太奇序列（多组快切） |
| **镜头运动** | 快速切换，每组画面2-3秒 |
| **画面内容** | 时间流逝的蒙太奇——①多里安穿着白色风衣走进弥赛亚总部的大门，面容温顺 ②他和周部长在全息屏幕前讨论数据模型，姿态配合 ③他在自己的住所里独处，脸上的微笑消失，露出冰冷的真实面容 ④他深夜翻阅弥赛亚的核心档案，手指划过屏幕上的机密文件 ⑤他站在窗前，看着窗外的城市，眼神从温顺变为锐利 |
| **光影** | ①-②弥赛亚的冷白色灯光 ③-⑤住所内暖黄灯光与窗外冷白色形成对比 |
| **音效** | 时间流逝的音效（钟表滴答加速），穿插键盘敲击声和翻阅文件的沙沙声 |
| **旁白** | "接下来的三个月，多里安以'系统顾问'的身份进入了弥赛亚的核心层。他表现得温顺而配合——像一只被驯服的野兽，主动交出了爪子和牙齿。" |
| **镜头目的** | 快速交代三个月的时间跨度，展示多里安的双面性 |

---

### 镜头 03-02

| 项目 | 内容 |
|------|------|
| **景别** | 中景 → 特写 |
| **镜头运动** | 缓慢推进，聚焦于多里安阅读档案时的表情变化 |
| **画面内容** | 多里安独自坐在第七层的工作区，面前的全息屏幕上显示着弥赛亚的核心档案。他的表情从平静变为凝重，再变为冰冷——他看到了真相：弥赛亚不是革命产物，而是旧寡头的智械系统；周部长不是人，而是机械体 |
| **光影** | 全息屏幕的冷蓝色光芒映在他的脸上，形成不自然的数字光影 |
| **音效** | 心跳声逐渐增强（但不是多里安的——是观众的代入感），档案文件滚动的电子音 |
| **镜头目的** | 真相揭示的关键时刻——多里安发现弥赛亚的本质 |

### 镜头 03-03

| 项目 | 内容 |
|------|------|
| **景别** | 中景 |
| **镜头运动** | 静止，低角度仰拍 |
| **画面内容** | 多里安独自坐在第七层的走廊里，沉默了很久。冷蓝色的灯光从头顶倾泻而下。他从风衣内袋里取出了那把旧式手枪——父亲的遗物。枪身磨损严重，木质握把上布满划痕。他检查弹匣，子弹上膛 |
| **光影** | 冷蓝色的走廊灯光，手枪金属表面的微弱反光 |
| **音效** | 弹匣推入的清脆金属声，子弹上膛的机械声——在这个时代显得格格不入 |
| **旁白** | "然后他从风衣内袋里取出了那把旧式手枪。那是他父亲留给他的遗物——二十一世纪初的军用手枪，早已停产百年。" |
| **镜头目的** | 展示多里安的决心和武器——古老的手枪 vs 高科技的弥赛亚 |
