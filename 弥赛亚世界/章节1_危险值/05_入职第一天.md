## 场景五：入职第一天

**时间**：2156年 / 上午
**地点**：弥赛亚秩序课大楼 → 第一第47层办公室
**时长估算**：约4分钟
**氛围**：新奇、紧张、人物群像展示

---

### 镜头 05-01

| 项目 | 内容 |
|------|------|
| **景别** | 大远景 → 缓慢推进 |
| **镜头运动** | 低角度仰拍，从街道地面缓慢抬起，跟随苏铭哲的视线向上 |
| **画面内容** | 苏铭哲站在公安局秩序课大楼前，仰头看向那座通体银白的建筑。大楼像一根插入云层的针，表面覆盖着流动的光膜，每隔几秒便有一次微弱的脉动，仿佛整栋楼本身就是一颗活着的心脏。苏铭哲的身影在建筑的映衬下极其渺小，他手腕内侧的植入芯片微微发光 |
| **光影** | 清晨的自然光与大楼表面的冷白色光膜交织，建筑投下的阴影覆盖了苏铭哲的上半身 |
| **音效** | 城市环境的白噪音，大楼内部传来的低频嗡鸣（弥赛亚系统的运转声），苏铭哲脚步声的轻微回响 |
| **旁白** | （男声，低沉）"2156年的天空是干净的。没有烟囱，没有尾气。弥赛亚系统接管世界的第一百年，人类终于学会了安分守己——或者说，学会了被教会安分守己。" |
| **镜头目的** | 建立弥赛亚建筑的压迫感，展示主角与系统的关系——渺小个体 vs 巨型机构 |

### 镜头 05-03

| 项目 | 内容 |
|------|------|
| **景别** | 中景 → 特写 |
| **镜头运动** | 电梯门打开，镜头跟随苏铭哲的脚步进入办公室，缓慢环视 |
| **画面内容** | 电梯门打开的瞬间，一股旧书的味道扑面而来。办公室中央是一张长桌，靠窗的位置最引人注目——一个高瘦的男人坐在那里，面前摊着一本厚重的纸质书。深蓝色的短发微乱，琥珀色的眼睛带着倦怠与忧郁。左手边摞着两本书，右手边放着一把旧式手枪和一把短刀。他的制服领口松着，整个人靠在椅背上，像一头疲惫的猎豹 |
| **光影** | 窗外的自然光从凯文身后打来，形成柔和的逆光，书页上的字迹在光线中清晰可见 |
| **音效** | 翻书页的沙沙声，办公室的低频环境音 |
| **对白** | 赵青（小声）："那就是凯文，你的搭档。两年前是指挥官，后来出了点事……他的危险值飙到过280。差点就回不来了。" |
| **镜头目的** | 第一次展示凯文——颓废、危险、带着古老的武器，建立"旧时代猎人"的形象 |

| **镜头目的** | 引入韩青云——严谨、冷硬、典型的弥赛亚体制内人物 |
