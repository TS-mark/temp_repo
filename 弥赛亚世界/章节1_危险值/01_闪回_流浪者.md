# 第一章　危险值 — 分镜脚本

## 场景一：弥赛亚发现多里安（闪回）

**时间**：两年前 / 夜晚
**地点**：城市边缘废弃工业区 → 弥赛亚监控中心
**时长估算**：约3分钟
**氛围**：孤独、神秘、压迫

---

### 镜头 01-01

| 项目 | 内容 |
|------|------|
| **景别** | 大远景 → 缓慢推进 |
| **镜头运动** | 无人机俯拍，从城市天际线缓缓下降，越过光鲜的摩天大楼群，逐渐靠近城市边缘的灰色地带 |
| **画面内容** | 城市的光鲜与边缘的荒芜形成强烈对比。左半边是弥赛亚管辖下的银白色建筑群，流光溢彩；右半边是废弃的工业区，锈蚀的钢梁和破碎的混凝土 |
| **光影** | 左侧城市冷白色调，右侧废弃区仅有月光和零星的火光 |
| **音效** | 城市方向传来低沉的电子嗡鸣（弥赛亚系统的运转声），逐渐被风声和金属吱嘎声取代 |
| **旁白** | （男声，低沉）"弥赛亚第一次注意到他，是在一次例行的数据扫描中。" |
| **镜头目的** | 建立世界观的空间对比——完美城市 vs 被遗忘的边缘 |

---

### 镜头 01-02

| 项目 | 内容 |
|------|------|
| **景别** | 中景 |
| **镜头运动** | 手持跟拍，微微晃动，跟随一个身影穿过废弃的管道 |
| **画面内容** | 多里安的背影。银白色短发在黑暗中隐约发光，白色风衣的下摆拖过积水的地面。他穿行在废弃的地下管道中，动作轻盈而警觉，像一只在阴影中移动的猫 |
| **光影** | 极暗，仅有管道裂缝中渗入的月光，在多里安的银发上形成一道微弱的光带 |
| **音效** | 脚步踩在积水上的轻响，远处金属结构的低频共振 |
| **镜头目的** | 首次展示多里安的外形轮廓，建立"幽灵"般的存在感 |

---

### 镜头 01-03

| 项目 | 内容 |
|------|------|
| **景别** | 特写 → 拉开至中景 |
| **镜头运动** | 从多里安的眼睛特写开始，缓慢拉开展示他的上半身 |
| **画面内容** | 多里安的脸。近乎透明的浅色眼睛，面容英姿俊俏得近乎妖异，嘴角挂着一丝淡淡的微笑。他坐在废弃工业区的高处，双腿悬空，俯瞰着远处的城市灯光 |
| **光影** | 月光从侧面打在他的脸上，一半明亮一半阴影，形成强烈的明暗对比 |
| **音效** | 夜风呼啸，远处城市的电子嗡鸣 |
| **镜头目的** | 第一次完整展示多里安的面容——美丽而冰冷 |

---

### 镜头 01-04

| 项目 | 内容 |
|------|------|
| **景别** | 界面特写 |
| **镜头运动** | 静止，画面叠加数字界面效果 |
| **画面内容** | 弥赛亚的监控界面。无数数据流在屏幕上滚动，突然一个红色的异常标记闪烁——一个无法被分类的个体。数据显示：人类男性，约25岁，危险值：0。不是"低"，是"零"。系统反复扫描，结果始终相同 |
| **光影** | 冷蓝色的界面光，红色异常标记的闪烁 |
| **音效** | 键盘敲击声、数据处理的电子音、异常警报的短促蜂鸣 |
| **旁白** | "系统的异常检测模块标记了一个无法被分类的个体——危险值始终为零。绝对的、毫无波动的零。" |
| **镜头目的** | 展示弥赛亚系统的工作方式，突出"零"的异常性 |

---

### 镜头 01-05

| 项目 | 内容 |
|------|------|
| **景别** | 大特写 |
| **镜头运动** | 缓慢推进，聚焦于多里安的眼睛 |
| **画面内容** | 多里安的眼睛。瞳孔没有收缩与扩张，没有情绪的波动。画面叠化：童年时期的多里安在笑、在哭、在为一只死去的鸟而难过——然后这些画面像灯一样一盏一盏熄灭，最终只剩下空白 |
| **光影** | 回忆画面暖色调，逐渐变冷，最终归于纯白 |
| **音效** | 儿童的笑声渐弱，最终归于寂静 |
| **旁白** | "他曾经有过情绪。但在某个时刻，他选择将这一切关闭了。不是压抑。是关闭。就像关掉一盏灯。" |
| **镜头目的** | 揭示多里安"零共鸣体"的本质——主动关闭，非天生缺失 |

---

### 镜头 01-06

| 项目 | 内容 |
|------|------|
| **景别** | 俯拍远景 |
| **镜头运动** | 从弥赛亚总部大楼的外景缓慢下降，穿过建筑外壁，进入内部 |
| **画面内容** | 弥赛亚总部大楼——银白色的巨型建筑，表面覆盖着流动的光膜。镜头穿透建筑外壁（CG过渡），进入一个巨大的数据处理中心。无数全息屏幕悬浮在空中，显示着多里安的档案数据 |
| **光影** | 冷蓝色为主，数据屏幕的光芒映照在光滑的金属地板上 |
| **音效** | 服务器运转的低频嗡鸣，数据流动的电子音效 |
| **旁白** | "弥赛亚的高层——那些隐藏在算法背后的旧寡头们的数字幽灵——做出了一个前所未有的决定：不清除他，而是吸收他。" |
| **镜头目的** | 展示弥赛亚的规模与力量，以及它对多里安的兴趣 |
