## 场景七：标本案件简报

**时间**：劫持事件后第三天 / 凌晨
**地点**：第一第47层办公室
**时长估算**：约3分钟
**氛围**：压抑、恐惧、旧案重提

---

### 镜头 07-01

| 项目 | 内容 |
|------|------|
| **景别** | 特写 → 中景 |
| **镜头运动** | 从数据板摔在桌上的特写快速拉开，展示整个办公室 |
| **画面内容** | 韩青云把数据板摔在桌上，发出清脆的撞击声。办公室里所有人都停下了手中的动作。全息投影亮起，显示出案发现场的照片。苏铭哲的胃猛地收缩了一下——那不是一具尸体，那是一件作品。死者的皮肤被某种生化制剂溶解，露出底下光滑的肌肉组织和骨骼结构，被摆成了一个跪地仰首的姿势，双手合十 |
| **光影** | 全息投影的冷蓝色光芒照亮所有人的面部，尸体照片的细节在光线下格外清晰 |
| **音效** | 数据板摔在桌上的撞击声，全息投影启动的电子音，苏铭哲呼吸的短暂停滞 |
| **对白** | 韩青云："昨晚，东区废弃工业带发现一具尸体。死者身份已确认：李维安，34岁，前高级程序员。三年前因危险值超标被分配至废料处理厂。" |
| **镜头目的** | 建立案件的视觉冲击——"艺术化的死亡"，暗示凶手的特殊心理 |
