# 第三章 疯狂王子的回归 — 分镜脚本

## 场景一：弥赛亚纪念广场——面具尸体

**时间**：清晨 / 林雨桐案后第四十七天
**地点**：弥赛亚纪念广场 / 银色尖塔
**时长估算**：约4分钟
**氛围**：肃穆、挑衅、神圣与亵渎的交织

---

### 镜头 03-01

| 项目 | 内容 |
|------|------|
| **景别** | 大远景 → 缓慢仰拍推进 |
| **镜头运动** | 从广场地面的低位机位开始，缓慢仰拍上升，沿尖塔向上推进 |
| **画面内容** | 弥赛亚纪念广场的清晨。二十米高的银色尖塔矗立在广场中央，表面覆盖着流动的光膜，在晨光中反射出冷冽的银白色。尖塔底部摆放着市民献上的合成花环——阶级革命纪念日的遗迹。镜头沿尖塔缓慢上升，越过花环，越过塔身的铭文——直到半腰处。一个悬挂在碳纤维丝上的人形映入画面 |
| **光影** | 清晨的冷白色天光，尖塔表面的光膜反射出流动的银色光泽。半腰处的尸体被晨光从下方照亮，形成不自然的阴影 |
| **音效** | 风声，碳纤维丝在风中微微振动的高频嗡鸣，远处城市苏醒的低频环境音 |
| **镜头目的** | 用仰拍的缓慢上升建立悬念——从纪念广场的"神圣"空间逐步揭示亵渎性的尸体 |
