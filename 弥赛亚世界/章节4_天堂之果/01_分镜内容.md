# 第四章　天堂之果 — 分镜脚本

## 场景一：猎场邀请

**时间**：夜晚
**地点**：秩序课办公室 / 数据终端前
**时长估算**：约3分钟
**氛围**：压抑、震惊、愤怒

---

### 镜头 04-01-01

| 项目 | 内容 |
|------|------|
| **景别** | 界面特写 → 缓慢拉至中景 |
| **镜头运动** | 从全息投影的数据界面开始，字符逐行解密显现，缓慢拉开展示围坐在终端前的众人 |
| **画面内容** | 赵辰的解密程序在全息投影上运行，加密字符像虫子一样蠕动、重组，最终排列成一段令人毛骨悚然的文字。镜头拉开，苏铭哲、凯文、韩青云、沈若水等人的脸在全息光芒中逐渐显现，表情各异 |
| **光影** | 全息投影的冷蓝色光映照在众人面部，办公室其余部分沉入暗影 |
| **音效** | 数据解密的电子滴答声，字符重组时的微弱嗡鸣，解密完成时一声清脆的提示音 |
| **旁白** | "线索来自多里安留在图书馆终端机里的最后一段数据——一段加密的邀请函。" |
| **镜头目的** | 建立信息揭示的节奏，从技术过程过渡到内容的冲击 |

## 场景二：凌晨四点的仙客来城堡

**时间**：凌晨4:00 / 深夜至黎明前
**地点**：废弃工业区 → 仙客来城堡（全息森林覆盖的钢铁厂）
**时长估算**：约3分钟
**氛围**：诡异、虚实交错、美丽与恐怖并存

---

### 镜头 04-02-01

| 项目 | 内容 |
|------|------|
| **景别** | 大远景 → 缓慢推进 |
| **镜头运动** | 无人机俯拍，从高空缓慢下降，靠近三座废弃钢铁厂的轮廓 |
| **画面内容** | 凌晨四点的城市边缘。三座废弃的钢铁厂在夜色中像三具巨大的骸骨，锈蚀的钢梁在月光下泛着暗红色的光。连接它们的管道网络如同血管般蜿蜒在地表。远处城市的灯光在地平线上形成一条冷白色的光带 |
| **光影** | 深蓝色夜空，月光为钢铁厂轮廓镀上银边，锈蚀部分呈暗红色，城市方向的冷白光微微映亮云层底部 |

| **镜头目的** | 建立仙客来城堡的真实面貌——废墟，与后续的全息幻象形成反差 |
