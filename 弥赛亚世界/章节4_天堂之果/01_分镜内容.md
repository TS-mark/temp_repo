**时间**：夜晚
**地点**：秩序课办公室 / 数据终端前
**时长估算**：约3分钟
**氛围**：压抑、震惊、愤怒

### 镜头 04-01-01

| 项目 | 内容 |
|------|------|
| **景别** | 界面特写 → 缓慢拉至中景 |
| **镜头运动** | 从全息投影的数据界面开始，字符逐行解密显现，缓慢拉开展示围坐在终端前的众人 |
| **画面内容** | 全息投影悬浮在办公桌上方约三十厘米处，投射出一片密密麻麻的加密字符——那些字符像是活物一样在空中蠕动、扭曲、重组，每一个字母和数字都在以不同的速度闪烁，仿佛在挣扎着挣脱某种束缚。赵辰的解密程序正在运行，进度条在屏幕底部缓缓推进，每前进一格，就有更多的字符被"解开"——它们停止蠕动，安静地排列成可读的文字。镜头开始缓慢拉远，首先露出苏铭哲的侧脸——他的眼睛紧盯着投影，瞳孔中映着蓝色的数据流，嘴唇微微抿紧。继续拉远，凯文的身影出现在画面右侧，他的双手交叉放在胸前，下巴微微抬起，琥珀色的眼睛半眯着，像是在评估什么。韩青云坐在苏铭哲左侧，他的眼镜反射着全息投影的蓝光，镜片后面的眼睛在快速扫视着逐渐成形的文字。沈若水的全息通讯影像在另一端微微闪烁，她的手指在虚拟键盘上快速敲击，调出更多的数据。办公室的其余部分沉入黑暗，只有全息投影的蓝光和几盏应急灯的微弱红光 |
| **光影** | 全息投影的冷蓝色光映照在众人面部，办公室其余部分沉入暗影 |
| **音效** | 数据解密的电子滴答声，字符重组时的微弱嗡鸣，解密完成时一声清脆的提示音 |
| **旁白** | "线索来自多里安留在图书馆终端机里的最后一段数据——一段加密的邀请函。" |
| **镜头目的** | 建立信息揭示的节奏，从技术过程过渡到内容的冲击 |

### 镜头 04-01-02

| 项目 | 内容 |

| **画面内容** | 全息投影上的加密字符终于完全解密，它们安静地悬浮在空中，排列成一封完整的邀请函。镜头推近至文字特写——第一行字缓缓亮起："仙客来城堡诚邀阁下莅临。"字体是古典的衬线体，带着一种维多利亚时代的优雅。第二行字紧随其后亮起："在这里，您将体验人类最古老的娱乐——狩猎。"每一个字都像是用墨水在羊皮纸上书写，带着冰冷的仪式感。第三行字开始显现："不同的是，我们的猎物不是鹿，不是熊……"字与字之间有微妙的停顿，像是在给读者时间消化。最后一行字缓缓浮现："我们的猎物，是人。"这个"人"字在出现的瞬间从白色变成了暗红色，像是被鲜血浸染。整个邀请函悬浮在空中，每一个字都散发着冰冷的、优雅的光芒 |

| **镜头目的** | 以文字本身的力量制造冲击，让观众逐字阅读并感受其恐怖 |

| **光影** | 深蓝色夜空，月光为钢铁厂轮廓镀上银边，锈蚀部分呈暗红色，城市方向的冷白光微微映亮云层底部 |
| **音效** | 夜风穿过废弃钢梁的呜咽声，远处金属结构的低频共振，偶尔的金属吱嘎声 |
| **旁白** | "仙客来城堡不是一座城堡。它是旧城区东缘的一片废弃工业区——三座废弃的钢铁厂、一座倒塌的化工车间、以及连接它们的、长达两公里的地下管道网络。" |
