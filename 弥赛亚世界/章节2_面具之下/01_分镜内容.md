# 第二章 面具之下 — 分镜脚本

## 场景一：废弃画廊——第三具尸体

**时间**：白天 / 下午
**地点**：城东废弃画廊
**时长估算**：约4分钟
**氛围**：诡异、压抑、令人窒息的"美感"

---

### 镜头 02-01

| 项目 | 内容 |
|------|------|
| **景别** | 大远景 → 中景推进 |
| **镜头运动** | 高位固定机位，缓慢推进。镜头从画廊建筑外景开始，穿过警戒线，进入画廊内部 |
| **画面内容** | 废弃画廊的外观——斑驳的白色外墙，破碎的玻璃窗，入口处拉着黄色警戒线。几名警员在外围把守。镜头缓缓推进穿过正门，进入画廊内部。空气中弥漫着一层淡淡的雾气，甜腻的化学溶剂气味仿佛能透过画面传达 |
| **光影** | 外景阴沉的天光，内部冷白色的人工照明，两种光源在门口形成交界 |
| **音效** | 风声，警戒线在风中拍打的声响，远处城市的低频嗡鸣。进入画廊后，声音突然变得压抑，只剩下空调系统运转的微弱嗡嗡声 |
| **镜头目的** | 建立空间——废弃画廊的荒凉外观与内部诡异的"展览"氛围形成反差 |

---

### 镜头 02-02

| 项目 | 内容 |
|------|------|
| **景别** | 中景跟拍 |
| **镜头运动** | 手持跟拍苏铭哲的背影，微微晃动，模拟主观视角。跟随他跨过警戒线，走向韩青云 |
| **画面内容** | 苏铭哲的背影。他穿着深色外套，步伐稳健但略显僵硬。跨过警戒线时脚步微微一顿——他闻到了那股气味。镜头跟随他穿过画廊前厅，看见韩青云站在画廊中央，面色铁青，镜片后的目光冷得像两片刀刃 |
| **光影** | 冷白灯光从头顶直射，在地面上投下清晰的人影 |
| **音效** | 脚步声在空旷的画廊中回响，苏铭哲的呼吸声略微加重 |
| **对白** | 韩青云（不回头，声音压低）："过来。" |
| **镜头目的** | 通过跟拍建立观众与苏铭哲的同步视角，制造"即将看见"的悬念 |

### 镜头 02-03

| 项目 | 内容 |
|------|------|
| **景别** | 中景 → 缓慢横移全景 |
| **镜头运动** | 从苏铭哲的侧面中景开始，他停住脚步。然后镜头缓慢向右横移，依次展示墙上的三件"作品" |
| **画面内容** | 苏铭哲的侧面——他的表情从困惑变为震惊，胃部微微翻涌，他勉强压下呕吐的冲动。镜头横移，依次展示三具被溶解、重塑后凝固在白墙上的人体。肢体被拉伸成流畅的曲线，躯干被切割成几何形状，面部表情定格在超越痛苦与宁静之间的状态。冷白灯光下，这些"作品"散发着诡异的、令人无法移开目光的美感 |
| **光影** | 冷白顶光，"作品"表面呈现出半透明的质感，投射出细碎的阴影 |
| **音效** | 音乐完全消失，只剩下呼吸声和空调的低频嗡鸣。苏铭哲的胃部翻涌声（主观音效） |
| **镜头目的** | 首次展示"艺术品"的全貌，用缓慢横移迫使观众逐一审视，强化冲击力 |

### 镜头 02-04

| 项目 | 内容 |
|------|------|
| **景别** | 特写 |
| **镜头运动** | 固定机位，依次切换三个特写画面 |
| **画面内容** | 第一个特写：一件"作品"的面部——表情被定格在某种超越痛苦与宁静之间的状态，嘴唇微张，仿佛在无声地叹息。第二个特写："作品"旁精心放置的书籍封面——赫胥黎的《美丽新世界》。第三个特写：另一件"作品"旁的奥威尔的《一九八四》 |
| **光影** | 微距灯光，书脊上的字迹清晰可见，"作品"皮肤的半透明质感被放大 |
| **音效** | 纸张翻动的细微声响（画外），低频心跳声渐起 |
| **对白** | 韩青云（画外）："这是第三起了。三天，三具，三个不同的地点。手法完全一致。" |
| **镜头目的** | 用特写建立"作品"与书籍的并置关系，暗示凶手的"创作意图" |

|------|------|
| **景别** | 中景对话 |

| **镜头目的** | 传递案件信息，建立受害者画像——被系统标记的"潜在违规者" |

### 镜头 02-06

| **光影** | 冷白灯光打在书的封面上，反射出微微的光泽。周围地面的反射形成一种"展览橱窗"的效果 |
| **音效** | 寂静。只有远处警员通讯器偶尔传来的杂音 |
| **镜头目的** | 本场景的核心画面——书作为凶手的"签名"，暗示这是一场针对弥赛亚的隐喻性抗议 |

### 镜头 02-07

| 项目 | 内容 |
|------|------|
| **景别** | 大特写 |
| **镜头运动** | 缓慢推进，聚焦于苏铭哲的眼睛 |
| **画面内容** | 苏铭哲的眼睛。瞳孔微微收缩，目光从书的封面上移，缓缓抬头看向墙上的"作品"，再低头看向书。他的表情——不是恐惧，不是厌恶，而是某种更深层的、他尚无法命名的震动。画面叠化：他入职前背诵的条例文字在脑海中闪回——"预防性安全""标准化思维模式" |

## 场景二：林雨桐身份调查

**时间**：调查进行到第三十六小时 / 傍晚
**地点**：秩序课办公室 / 数据分析室
**时长估算**：约3分钟
**氛围**：紧张、压抑、暗示性的危险

---

### 镜头 02-08

| 项目 | 内容 |
|------|------|
| **景别** | 界面特写 → 拉开至中景 |
| **镜头运动** | 从全息屏幕的数据界面特写开始，缓慢拉开展示苏铭哲坐在数据分析台前的全貌 |
| **画面内容** | 全息屏幕上数据流飞速滚动，苏铭哲的手指在虚拟键盘上快速敲击。屏幕上显示着社交关联分析图谱，无数节点和连线在三维空间中旋转。突然，一个被红色边框标注的档案浮出——"身份存疑"。镜头拉开，展示苏铭哲疲惫但专注的面容，桌上散落着数据芯片和冷掉的合成咖啡 |
| **光影** | 全息屏幕的冷蓝光映在苏铭哲脸上，形成明暗交替的光斑 |
| **音效** | 键盘敲击的电子音，数据处理的嗡鸣，全息界面切换的电子音效 |
| **镜头目的** | 展示调查过程的技术感，用"身份存疑"的红色标记制造悬念 |

### 镜头 02-09

| 项目 | 内容 |

## 场景三：林雨桐的工作室

**时间**：调查期间 / 白天
**地点**：城郊即将拆迁的居民楼顶层
**时长估算**：约4分钟
**氛围**：震惊、诡异、神圣与恐怖的交织

---

### 镜头 02-14

| 项目 | 内容 |
|------|------|
| **景别** | 大远景 → 中景推进 |
| **镜头运动** | 从居民楼外景开始，缓慢上升推进至顶层 |
| **画面内容** | 城郊一栋即将拆迁的居民楼。外墙斑驳，窗户大多破碎，脚手架已经搭了一半。镜头从地面缓慢上升，越过一层层破败的楼层，最终抵达顶层——唯一亮着灯的窗户。窗帘紧闭，隐约可见内部的光亮 |
| **光影** | 阴天的灰白色天光，顶层窗户透出的暖黄色灯光形成唯一的暖色 |
| **音效** | 风声，远处拆迁工地的机械声，居民楼结构在风中吱嘎作响 |
| **镜头目的** | 建立林雨桐工作室的外部环境——被遗忘的边缘地带 |

---

### 镜头 02-15

| 项目 | 内容 |
|------|------|
| **景别** | 主观视角 → 中景 |
| **镜头运动** | 沈若水的主观视角推开门，然后切换为中景展示工作室全貌 |
| **画面内容** | 门被推开。主观视角：首先映入眼帘的是一间普通的画室——画架、颜料、未完成的素描、散落的速写本。然后目光越过画室，看见后半部分——一扇半开的门通向一个密闭的、温度恒定的房间。切换为中景：工作室被分为两个截然不同的空间，前半部是画室的温暖凌乱，后半部是密闭空间的冰冷秩序 |
| **光影** | 前半部画室：自然光从窗户渗入，暖黄色调。后半部密闭空间：纯白色墙壁反射的冷白光 |
| **音效** | 门推开的吱嘎声，画室内颜料和画布的气味（通过沈若水的呼吸声暗示），密闭空间传出的恒温设备的低频嗡鸣 |
| **对白** | 沈若水（画外，语气平静）："我以为我会看见画布、颜料、石膏像。但我看见的是一个屠宰场，和一个神殿。" |
| **镜头目的** | 用空间的二元对立建立核心冲突——艺术创作与人体处理的共存 |

---

### 镜头 02-16

| 项目 | 内容 |
|------|------|
| **景别** | 中景 → 缓慢环绕 |
