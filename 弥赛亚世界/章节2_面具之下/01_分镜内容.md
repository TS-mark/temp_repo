# 第二章 面具之下 — 分镜脚本

## 场景一：废弃画廊——第三具尸体

**时间**：白天 / 下午
**地点**：城东废弃画廊
**时长估算**：约4分钟
**氛围**：诡异、压抑、令人窒息的"美感"

---

### 镜头 02-01

| 项目 | 内容 |
|------|------|
| **景别** | 大远景 → 中景推进 |
| **镜头运动** | 高位固定机位，缓慢推进。镜头从画廊建筑外景开始，穿过警戒线，进入画廊内部 |
| **画面内容** | 镜头从画廊建筑的正对面开始拍摄，画廊的白色外墙在阴沉的天光下显得格外斑驳——墙皮大面积脱落，露出底下灰黑色的混凝土基底，有几处涂鸦已经被雨水冲刷成模糊的色块。正门两侧的玻璃窗全部碎裂，碎玻璃茬子在窗框上参差不齐地排列着，像一排断裂的牙齿。入口处拉着三道黄色警戒线，警戒线在微风中轻轻拍打着门框，发出塑料碰触金属的细微声响。两名警员站在门外，双手背在身后，面部表情僵硬而疲惫，其中一人偶尔抬手看一眼手腕上的时间。镜头缓缓向前推进，穿过警戒线之间的缝隙进入画廊内部——首先映入眼帘的是一片昏暗的前厅，地面上覆盖着一层薄薄的灰尘，空气中弥漫着一层淡淡的、近乎可见的雾气，那股甜腻的化学溶剂气味仿佛能透过画面传达过来，让人不自觉地皱起鼻子 |
| **光影** | 外景阴沉的天光，内部冷白色的人工照明，两种光源在门口形成交界 |
| **音效** | 风声，警戒线在风中拍打的声响，远处城市的低频嗡鸣。进入画廊后，声音突然变得压抑，只剩下空调系统运转的微弱嗡嗡声 |
| **镜头目的** | 建立空间——废弃画廊的荒凉外观与内部诡异的"展览"氛围形成反差 |

| **镜头目的** | 首次展示"艺术品"的全貌，用缓慢横移迫使观众逐一审视，强化冲击力 |
