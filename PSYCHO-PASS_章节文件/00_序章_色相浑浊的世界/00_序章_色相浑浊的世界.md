# 序章：色相浑浊的世界

西元2112年。

人类终于找到了通往"乌托邦"的道路——至少，所有人都被如此告知。

那是一个没有战争、没有饥荒、没有贫穷的时代。基因工程消除了大部分遗传疾病，人工智能接管了所有重复性劳动，物质的丰裕让每一个公民都能过上体面的生活。但人类最大的敌人——犯罪——却始终无法被根除。直到"西比拉系统"的诞生。

这个庞大的神经网络从诞生之日起便接管了整个日本列岛的社会管理。它无处不在——嵌入墙壁中的传感器、悬浮在空中的微型无人机、植入每一个公共设施的扫描终端——它们如同一张无形的网，将每一个人包裹其中。西比拉系统能实时读取一个人的精神压力、情绪波动、犯罪倾向，并将这些复杂的心理数据转化为一组简洁而精确的数值。

其中最重要的一个数字，便是"犯罪系数"（Crime Coefficient）。

当犯罪系数低于100时，你是良民——你的色相（Hue）清澈，你的心理状态稳定，你是这个完美社会中一个完美的零件。
当它升至100至300之间时，你会被警告、被隔离、被"治疗"——你的色相开始浑浊，你正在滑向危险的深渊。
当它突破300的那一刻——你将不再是公民，而是猎物。公安局的执行官会被派出，手持"主宰者"（Dominator），将你从这个世界上抹除。

在这样的社会中，人们不需要做选择——西比拉系统会为你选择。它决定你适合什么职业，适合什么伴侣，适合什么样的生活。你的升学、就业、婚姻、甚至每天的出行路线，都在系统的"建议"之下进行。人们心甘情愿地交出了自己的自由，因为系统承诺了一个代价：安全。

公安局刑事课——一个由"监视官"和"执行官"组成的执法机构，负责处理那些犯罪系数超标的人。监视官是正常公民，经过选拔成为执法者，拥有完整的公民权利；而执行官，则是那些犯罪系数曾经超标、被判定为"潜在犯"的人——他们曾经是猎物，如今被赋予了猎杀同类的权力。他们的脚踝上嵌入了追踪芯片，他们的活动范围被严格限制，他们的每一个举动都被监视官所监控。

他们是武器，是工具，是被系统遗弃又被系统利用的人。

在这样的世界里，正义是什么？自由又是什么？当一个人的灵魂被简化为一个数字，当一个社会的秩序建立在对每一个人的心理监控之上——这样的"和平"，还值得守护吗？

这个故事，便始于一个对这个世界一无所知的少女，踏入公安局大楼的那个清晨。

她叫常守朱。

她将用自己的一生来回答这些问题。
